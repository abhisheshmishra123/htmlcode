HTML stands for Hyper Text Markup Language. It is standard and fundamental         markup language for creating web pages and websites. It specifies the structure of the web page. The building blocks of HTML are the elements of the HTML. HTML elements are represented by tags and their
attributes. Web browsers do not display the HTML tags,
but use them to render the contents of the web page. Web browsers receive HTML documents from web server or from local storage and render the document into web pages.
